# Extreme Events Classifier

XGBoost classifier for extreme weather event probability prediction.

## Model Architecture

- **Type**: XGBoost
- **Framework**: XGBoost
- **Input**: Climate features, anomaly indicators
- **Output**: Probabilities for heatwave, drought, flood, cold snap

## Performance

- Accuracy: 0.92
- Precision: 0.89
- Recall: 0.91
- F1 Score: 0.90
